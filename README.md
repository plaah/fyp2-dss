BPJS claim grouping outcomes and provide actionable recommendations
for hospital Casemix coders.

## Tech Stack
- Backend: Flask (Python)
- ML: XGBoost, LightGBM, SHAP
- Database: PostgreSQL + SQLAlchemy
- Frontend: HTML/CSS/JS + Chart.js
